# The Ying Markup Language

### Usage method:
1. #### touch a file named note.ing, like..
```
! This is first level title.
    @ This is second level title.
        # This is third level title.
            Here are some words.Here are some words.
            Here are some words.Here are some words.Here are some words.
```
2. #### explain it:
```shell
ying note.ing note.html
```
3. #### open the note.html with your chrome
